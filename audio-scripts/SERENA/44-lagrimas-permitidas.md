# 44 — Lágrimas Permitidas
**Módulo:** SERENA | **Tipo:** Prática de fluidez guiada | **Duração:** ~3 min
**Nível:** 1 (Nascente) | **ID:** `lagrimas-permitidas` | **Para:** Libertação emocional, permissão para sentir

**ElevenLabs:** Modelo V3 · Voz feminina portuguesa · Speed: 0.75 · Stability: 80

## Script (colar directamente no ElevenLabs)

[softly] Se estás a segurar lágrimas... [pause] este é o teu momento para as deixar cair. [long pause]

[gently] Não precisas de uma razão grande para chorar. [pause] Não precisas de justificar. [pause] Às vezes o corpo sabe antes de nós... [pause] e pede para soltar. [long pause]

Fecha os olhos. [pause] Põe uma mão no peito. [long pause]

[softly] Diz para ti mesma, em silêncio: [pause] Eu posso chorar. [pause] Chorar não é fraqueza. [pause] Chorar é a água a limpar o que transbordou. [long pause]

[gently] Se as lágrimas vierem... [pause] deixa-as. [pause] Não limpas logo. [pause] Não controles o rosto. [pause] Deixa o corpo fazer o que precisa. [long pause]

[long pause]

[softly] Cada lágrima leva um bocadinho de peso embora. [pause] Como chuva que limpa o ar. [pause] Depois da chuva... [pause] tudo fica mais nítido. [long pause]

[long pause]

[warmly] Se não choraste... [pause] tudo bem também. [pause] Só o facto de teres dado permissão já é fluidez. [long pause]

[gently] O teu coração sabe quando é altura. [pause] Tu confiaste nele ao estar aqui. [long pause]

[warmly] Respira fundo. [pause] Estás mais leve agora.
