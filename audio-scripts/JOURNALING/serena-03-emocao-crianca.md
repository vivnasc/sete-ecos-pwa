# Prompt de Journaling Guiado — A Primeira Vez
**Tipo:** Guided Audio Prompt | **Duração:** ~60-90s (~800-1200 caracteres)
**Eco:** Serena | **Secção:** Diário emocional — origem emocional
**Uso:** Tocado antes de escrever no diário/exercício. Botão ▶ no UI.

**ElevenLabs:** Modelo V3 · Voz Vivianne · Speed: 0.75 · Stability: 85

---

## Script (colar directamente no ElevenLabs)

[softly] Fecha os olhos. Respira fundo, uma vez.

[pause: 5s]

Pensa no que tens sentido ultimamente. Essa emoção que aparece, que volta, que insiste.

[pause: 3s]

[gently] Agora deixa a tua mente viajar para trás. Devagar. Como quem folheia um álbum antigo.

[pause: 3s]

Quando foi a primeira vez que sentiste isto? Consegues lembrar-te?

[pause: 5s]

Talvez estavas numa sala. Talvez estavas sozinha. Talvez eras muito pequena — tão pequena que nem tinhas palavras para o que sentias.

[pause: 3s]

[slowly] Quantos anos tinhas? Onde estavas? Quem estava perto de ti — ou quem não estava?

[pause: 5s]

Essa criança não sabia o que fazer com aquilo. Guardou. Escondeu. E carregou até aqui.

[pause: 3s]

[softly] Abre os olhos agora. E escreve para ela. Para a criança que sentiu isto pela primeira vez. Conta-lhe o que sabes hoje. O que dirias a essa versão mais nova de ti?

[pause: 3s]

Escreve com ternura.
