# Prompt de Journaling Guiado — A Carta Nunca Enviada
**Tipo:** Guided Audio Prompt | **Duração:** ~60-90s (~800-1200 caracteres)
**Eco:** Ecoa | **Secção:** Cartas nunca enviadas
**Uso:** Tocado antes de escrever no diário/exercício. Botão ▶ no UI.

**ElevenLabs:** Modelo V3 · Voz Vivianne · Speed: 0.75 · Stability: 85

---

## Script (colar directamente no ElevenLabs)

[softly] Fecha os olhos. Solta os ombros. Solta as mãos.

[pause: 3s]

Respira fundo... e ao expirar, imagina que abres uma gaveta antiga dentro de ti. Uma gaveta que guardas fechada há muito tempo.

[pause: 5s]

Lá dentro, há uma carta. Nunca a escreveste de verdade — mas ela existe. Dentro de ti. Cada palavra, cada vírgula, cada silêncio entre as linhas.

[pause: 3s]

[gently] Para quem é essa carta?

[pause: 5s]

[slowly] Talvez para alguém que te magoou. Talvez para alguém que partiu. Talvez para alguém que ainda está aqui — mas nunca ouviu o que precisavas de dizer.

[pause: 3s]

Não precisas de enviar esta carta. Ela não é para a outra pessoa. É para ti. Para tirares de dentro o que já não cabe.

[pause: 5s]

[softly] Abre os olhos. E escreve essa carta agora. Começa por: "Precisava de te dizer..." — e deixa a mão guiar-te. Sem censura. Sem rascunho.

[pause: 3s]

Esta carta é tua. Só tua.
