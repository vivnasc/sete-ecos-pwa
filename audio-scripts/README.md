**Total: ~100 áudios a produzir** | Actualizado: 12 de Março de 2026

## Como usar este repositório

Cada ficheiro `.md` contém:
- Metadados do áudio (módulo, duração, tipo, nível de acesso)
- **Script completo** para leitura/gravação (colar directamente no ElevenLabs)
- Notas de entoação e direcção vocal

**Voz**: Vivianne — feminina, suave, ritmo lento. Pausas respeitadas.
**Língua**: Português (PT/MZ) — manter acentos sempre.
**Formato final de ficheiro**: `.mp3` estéreo, 44.1kHz, 128kbps mínimo.
**ElevenLabs**: Modelo V3 · Voz Vivianne · Speed variável por tipo

## Depois de gravar — o que fazer

### ÁUREA
1. Upload `.mp3` para Supabase Storage
2. Abrir `src/components/aurea/AudioMeditacoes.jsx`
3. Substituir `audio_url: null` pela URL de cada áudio

### IMAGO
1. Upload e actualizar `MeditacoesEssencia.jsx` (requer desenvolvimento)

### SERENA
1. Requer alteração no componente `RespiracaoGuiada.jsx`

### LUMINA
1. Upload para Supabase Storage (pasta `lumina-audio/`)
2. Mapeamento padrão→URL em constante no código
3. Botão ▶ aparece na tela de leitura ao lado do texto

### WHATSAPP
1. Upload como Media no Meta Business Suite
2. Integrar media_id no `api/_lib/wa-sequencia-cron.js`

### PODCAST
1. Upload para host (Supabase Storage ou Spotify for Podcasters)
2. Criar secção "Vivianne Explica" na app (explorar/comunidade)
3. Recortar clips de 15-30s para audiogramas/Reels

### JOURNALING
1. Upload para Supabase Storage
2. Adicionar botão ▶ nos componentes de diário (Serena, Ecoa, Imago)

### AUDIOGRAMAS
1. Gerar com ferramenta de audiograma (Headliner, Wavve, ou Canva)
2. Publicar via `api/_lib/instagram-schedule.js`
